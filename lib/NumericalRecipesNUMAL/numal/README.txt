All the source code of the book are included On this CD.

The source code are ASCII files with file extension "java".

On the CD, the files whose names have file extension "class"
are platform-independent bytecodes produced from the source
code by the java compiler: 
   Java 2 SDK, Standard Edition, version 1.4.1.

The ASCII *.java files should be readable by any platform and does not require any particular software to read them.  The *.class files can be reproduced for any other platform by re-compiling the ASCII *.java files using any Java compiler.
